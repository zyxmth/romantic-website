# Romantic Website

This project is a modern, responsive website designed with a romantic red and rose color theme. It features a clean and emotional layout, perfect for sharing heartfelt messages.

## Project Structure

```
romantic-website
├── index.html          # Main entry point of the website
├── css
│   └── styles.css     # Styles for the website
├── js
│   ├── script.js      # Main JavaScript functionality
│   └── hearts.js      # Animation for falling hearts
├── pages
│   ├── message1.html  # First message screen
│   └── message2.html  # Second message screen
├── package.json       # Configuration file for npm
└── README.md          # Documentation for the project
```

## Features

- **Responsive Design**: The website adapts seamlessly to mobile phones, tablets, and desktops.
- **Falling Hearts Animation**: Softly animated heart shapes fall from the top of the screen, enhancing the romantic theme without being distracting.
- **Interactive Buttons**: The main button and smaller buttons provide a smooth user experience with hover and tap effects.
- **Message Screens**: Two separate message screens that users can navigate to, each with a back button to return to the main page.

## Getting Started

1. **Clone the Repository**: 
   ```
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory**:
   ```
   cd romantic-website
   ```

3. **Open `index.html`**: Open the `index.html` file in your web browser to view the website.

4. **Dependencies**: This project does not require any additional dependencies beyond standard HTML, CSS, and JavaScript.

## Usage

- Click the main button to navigate to the first message screen.
- Use the smaller buttons to view different messages.
- Return to the main page using the back buttons on the message screens.
